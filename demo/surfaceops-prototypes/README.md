# SurfaceOps Prototype Review Set

These static prototypes are review artifacts for the future SurfaceOps product surface. They are not proof authority, generated evidence, live SurfaceOps, live JudgmentKit, production adapters, runtime behavior, API support, SDK support, A2UI support, or work-order execution.

Open `index.html` in this folder to review the four variants:

- Evidence Workbench
- Decision Flow
- Governance Board
- Contract Proof DAG Canvas

The prototypes follow the SurfaceOps product brief and UI criteria under `plans/`, with JudgmentKit used as the activity and surface-type guide.

The Contract Proof DAG Canvas explores the V2 visual proof direction: a React Flow based DAG where generation, integration, runtime, and evidence nodes drive a right-side inspector for the selected contract. It is still a static prototype and does not create proof authority.
